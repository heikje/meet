Overview

The Main feature of this module, is to give the possibility to create a Google Meets/Hangouts from Odoo Calendar and Synchronize it with Google Agenda linked to your Odoo account.

Features

  • Create Google online meeting Meets/Hangouts from Odoo Calendar Event
  • Synchronize Google Agenda with Odoo Calendar Event
  • Synchronize Odoo Calendar event from Google Agenda
  • Access To the Online Meeting from Odoo Calendar Event

Requirements

  • Configure Odoo with Google Agenda Tutorial
  • Synchronize Odoo with Google Agenda
